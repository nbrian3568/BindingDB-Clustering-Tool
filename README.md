# BindingDB-Clustering-Tool
Web based tool to cluster BindingDB ligands by UniProt ID.

Steps:
1) Upload a tsv file that has SMILE strings, affinity data, and UniProt IDs.
2) Click on Upload & Cluster Button and wait for the webpage to process the data from the file.
3) A dropdown that contains the UniProt IDs associated with that dataset should appear.
4) Select the UniProt ID of the protein you want.
5) Clusters will appear with the molecules in each cluster and affinity histograms will be displayed for each UniProt ID.
